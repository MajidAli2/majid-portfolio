# Majid Portfolio

Personal portfolio site for Majid Ali, built with React and Vite.

## Features

- Hero section with quick intro and social links
- About, skills, projects, journey, services, certifications, and contact sections
- Responsive layout with animated visual styling
- Resume download from the public folder

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- `public/CV_PDF.pdf` is used for the resume download link.
- Social profiles and contact details are configured in `src/App.jsx`.
